# Sound Booster 1000

Sound Booster 1000 is a powerful browser extension that amplifies audio on any website up to **1000%** (10x boost). It features a premium design, dark/light theme support, and instant presets for the best listening experience.

![Preview](icon.png)

## Features

- **10x Power:** Boost volume up to 1000%—far beyond standard browser limits.
- **Sleek UI:** Modern, premium interface with glassmorphism and smooth animations.
- **Dark Mode:** Automatically adjusts to your system theme or can be toggled manually.
- **Smart Presets:** Quick access to 3x, 6x, and MAX (10x) boost levels.
- **Real-time Control:** Precision slider to find your perfect volume.

## Installation

Currently, you can install the extension manually:

### Firefox
1. Download this repository as a ZIP and extract it.
2. Open Firefox and go to `about:debugging`.
3. Click on **"This Firefox"**.
4. Click **"Load Temporary Add-on..."**.
5. Select the `manifest.json` file in the project folder.

## Link
